## üì¶ Installation & Setup

Follow these steps to get a local copy of the project up and running.

### Prerequisites
* **Node.js** (v18 or higher recommended)
* **Angular CLI** (`npm install -g @angular/cli`)

### Steps

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/raymondoyondi/Job-Listings-App.git](https://github.com/raymondoyondi/Job-Listings-App.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd Job-Listings-App
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Start the development server:**
    ```bash
    ng serve
    ```

5.  **Open your browser:**
    Navigate to `http://localhost:4200/` to see the app in action!

## üèóÔ∏è Architecture

The project follows a modular Angular structure to ensure scalability and maintainability:

* **Components:** Modular UI elements (JobCard, FilterBar, Header) that manage specific sections of the view.
* **Services:** Centralized logic used for fetching data and handling the complex filtering logic across the app.
* **Models:** Strict TypeScript interfaces defining the shape of Job objects, ensuring data integrity.
